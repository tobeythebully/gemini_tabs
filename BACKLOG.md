## 🗂️ PARA 연동 기능 상세

### ✅ 완료 (v1.1.0)
- 설정에서 PARA 루트 폴더 지정
- Projects 하위 폴더 스캔 및 선택
- 💾 저장 버튼으로 대화를 마크다운 저장
- 프로젝트 선택됨 → `Projects/[프로젝트]/outputs/`
- 프로젝트 미선택 → `Resources/gemini/`

---

### 🔴 다음 구현 (v1.2.0) - PARA 저장 규칙 개선

**저장 위치 수동 선택 (Option A)**
```
저장 모달:
├── ○ Areas (내 경험/생각)
├── ○ Resources (외부 정보/리서치)  
└── ○ Projects (프로젝트 산출물)

하위 폴더:
└── [기존 폴더 드롭다운 선택]
```

| 항목 | 내용 |
|-----|------|
| 저장 위치 | 매번 Areas/Resources/Projects 중 선택 |
| 하위 폴더 | 기존 폴더 목록에서 드롭다운 선택 |

---
