## 🗂️ PARA 연동 기능 상세

### 개념
로컬 PARA 시스템 폴더와 연동하여:
1. **Projects** 폴더에서 프로젝트별 컨텍스트 자동 로드
2. **Resources** 폴더에서 프롬프트 템플릿 불러오기
3. Gemini 응답을 **Archives**에 자동 저장

### 구현 가능성
- ✅ Electron은 Node.js 기반 → `fs` 모듈로 로컬 파일 접근 가능
- ✅ 파일 감시 (`fs.watch`)로 실시간 동기화 가능
- ✅ 마크다운 파일 읽기/쓰기 가능

### 예시 사용 흐름
```
1. 설정에서 PARA 루트 폴더 지정: ~/Documents/PARA/
2. /project 명령어로 프로젝트 선택
3. 해당 프로젝트의 context.md 자동 로드
4. Gemini 응답 저장 시 해당 프로젝트 폴더에 저장
```
