# styles.css 개발 로그

## 파일 개요
- **역할**: 앱 스타일시트
- **테마**: 다크 모드 (모노크롬)

---

## 디자인 시스템

### 색상 팔레트
```css
--bg-primary: #1a1a1a;    /* 메인 배경 */
--bg-secondary: #2a2a2a;  /* 탭바, 모달 배경 */
--text-primary: #ffffff;   /* 기본 텍스트 */
--text-muted: #888888;     /* 보조 텍스트 */
--accent: #ffffff;         /* 강조 (활성 탭) */
```

### 주요 컴포넌트

| 클래스 | 용도 |
|--------|------|
| `.tab` | 탭 버튼 스타일 |
| `.tab.active` | 활성 탭 (밑줄 표시) |
| `.bookmark-item` | 북마크 아이템 |
| `.modal` | 설정 모달 오버레이 |
